No image · images/proj-emotion-robot.png

Physically built a 4-wheeled Arduino robot with real-time facial expression detection. Designed and implemented an edge-cloud split architecture to solve a hard constraint: the Arduino lacks the compute for deep learning inference.

  • Edge-cloud split: CNN (4.47M params) runs on PC, transmits detected emotion over Bluetooth HC-06 to Arduino at 9600 baud
  • CNN trained on FER-2013 (28,709 images, 7 emotion classes) — 62% accuracy vs 14.3% random baseline
  • Haar Cascade face detection via OpenCV; motion sequences and speech synthesis per emotion class
  • Custom PCB layout, L298N dual H-bridge motor driver — graded 30/30 Cum Laude by examining panel
